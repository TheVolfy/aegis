# Aegis

Шифровальщик сообщений и файлов с графическим интерфейсом. Работает полностью оффлайн.

## Возможности

- Шифрование и дешифрование текстовых сообщений
- Шифрование и дешифрование файлов любого формата
- Двойное шифрование двумя независимыми алгоритмами
- Надёжная защита пароля от перебора (Argon2id)
- Уникальная соль для каждого сообщения — одинаковый текст с одним паролем даёт разные токены
- Встроенная проверка целостности данных (AEAD)

## Требования

- Python 3.10+
- Зависимости:

```bash
pip install argon2-cffi cryptography
```

## Запуск

### Графический интерфейс

```bash
python aegis_gui.py
```

### Командная строка

```bash
# Зашифровать сообщение
python aegis.py encrypt -m "Текст сообщения"

# Зашифровать с интерактивным вводом
python aegis.py encrypt

# Зашифровать файл
python aegis.py encrypt -f document.pdf

# Дешифровать сообщение
python aegis.py decrypt -t "токен..."

# Дешифровать файл
python aegis.py decrypt -f document.qc -o document.pdf

# Показать параметры шифрования
python aegis.py info
```

## Формат зашифрованных файлов

Зашифрованные файлы сохраняются с расширением `.qc` и содержат Base64url-текст. Файл можно безопасно передавать по email, мессенджерам и хранить в облаке.

## Алгоритмы

| Компонент | Алгоритм |
|---|---|
| Вывод ключа | Argon2id (128 МБ RAM, 4 итерации) |
| Шифрование (слой 1) | AES-256-GCM |
| Шифрование (слой 2) | ChaCha20-Poly1305 |
| Соль | 256 бит, случайная для каждого сообщения |

Безопасность основана исключительно на пароле — знание исходного кода не даёт никакого преимущества при попытке дешифровки.
